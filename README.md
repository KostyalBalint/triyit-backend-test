# node-starter

A quickstart repository for NodeJs projects

## Getting Started

To get started, install the dependencies:

```bash
yarn install
```

## Running the app

For automatic reloading with nodemon, run:

```bash
# development
yarn dev
```

## Test

We use Jest for testing. To run the tests, run the following command:

```bash
# unit tests
yarn test
```
